# Unsupervised-ML-Data-Mining
 Coursework from DS5230 Unsupervised ML &amp; Data Mining at Northeastern University — clustering, dimensionality reduction, topic modeling, recommender systems, and probabilistic   sampling in Python.

 | Notebook | Topic |
  |---|---|
  | academic-citation-network-analysis | Aminer dataset parsing, author/venue statistics, impact factor analysis |
  | kmeans-gaussian-mixture-em | Custom KMeans, Soft KMeans, Gaussian Mixture Models, EM algorithm |
  | dbscan-density-clustering | DBSCAN on spatial data with cluster visualization |
  | custom-dbscan-umap-high-dimensional | Custom DBSCAN + UMAP on MNIST, FashionMNIST, 20Newsgroups |
  | classification-logistic-regression-decision-trees | Logistic regression & decision trees on MNIST, SpamBase, 20Newsgroups |
  | tsne-dimensionality-reduction | t-SNE on MNIST followed by DBSCAN in reduced space |
  | word-embeddings-neural-networks-nlp | Word2Vec, neural networks, PCA on text data |
  | topic-modeling-lda-nmf | LDA & NMF topic modeling on news corpora |
  | probabilistic-sampling-gibbs-mcmc | Uniform, Gaussian, and Gibbs sampling — MCMC methods |
  | recommender-systems-community-detection | Collaborative filtering on MovieLens, Girvan-Newman community detection |
  | image-clustering-pca-kmeans | PCA + KMeans on noisy handwritten digit images |

  ## Stack
  Python · scikit-learn · NumPy · pandas · Matplotlib · TensorFlow · gensim · NetworkX
